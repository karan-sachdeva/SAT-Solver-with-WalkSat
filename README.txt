INSTRUCTIONS TO RUN
1. Run cnf.py on python 3.8.
2. Input your desired values for k,x and y and you will get a CNF file.
3. There's some example CNF files in the src folder.
4. Then run the java files situated in the WalkSat package in src folder.
5. The file will ask you the name of the CNF file.
6. The program will then return a solution if found.
7. Read the report for some insights.
